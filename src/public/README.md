# Public Code

This folder contains code that can be imported across both frontend pages and backend modules.

## Files

- `utils.js` - Shared utility functions
- `constants.js` - App-wide constants

## Usage

Import public code like this:

```javascript
import { formatPrice, showNotification } from 'public/utils';
```
